Sobre

O Python User Group do Vale do Paraíba (PUG-Vale) é uma iniciativa comunitária que tem o objetivo de reunir os desenvolvedores e demais interessados na linguagem de programação Python e em suas tecnologias associadas.

A finalidade do grupo é difundir o uso de Python na região, promovendo espaços e oportunidades para networking e troca de experiências.

O evento é gratuito, mas pedimos a sua contribuição com 1 quilo de feijão para que seja doado para uma instituição da região.
